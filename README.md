# console-snippets 
All JavaScript console snippets to save your time.

## Install
Press F1 and select `Extensions: Install Extensions`.
Search for and select `console-snippets`.

### Supported languages
- JavaScript
- TypeScript

## Snippets
- `cl` or `clog` console.log()
- `cwarn` console.warn()
- `cdir` console.dir()
- `ci` console.info()
- `cerr` console.error()
- `cg` console.group()
- `cge` console.groupEnd()
